Molecule Quilts

As an offshoot of my work creating renderings of molecules for my Molecules book, I developed Mathematica code to turn MOL files into various pretty forms, including nice sort-of-physics based glows around the structure diagrams. Naturally I decided to extend this code to create quilting and embroidery patterns.
